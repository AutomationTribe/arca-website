Account-to-account switch(ACH)

ACH is a system that focuses on initiating either a Credit
or Debit. It is an account number based, online real-time platform
that helps with the delivery of transaction messages between
participating financial institutions within seconds.

TRANSACTION SETS IN THE ACH

ACCOUINT ENQUIRY

DEBIT CARDS

DEBIT CREDIT

BALANCE QUERY

TRANSACTION QUERY

WHO IS IT FOR?

ACH is available for use for Financial Institutions e.g. Commercial banks, MFBs, Finance
houses etc
