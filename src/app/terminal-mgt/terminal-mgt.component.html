SERVICES IN THE TMS

AUTHORIZATION API

For issuing and verifying the identity of the users.

USER MANAGEMENT API

Used to create and manage all information around user

ACQUIRER MANAGEMENT API

Manages all information around acquirers

TRANSACTION SERVICE API

Captures enhanced payment data pushed from each terminal and write to a database for logging

TERMINAL MANAGEMENT API

Manages all information around POS terminals

MERCHANT MANAGEMENT API

Manages all information around merchants

DEVICE MONITORING

REMOTE DEVICE OPERATIONS

TRANSACTIONAL E-JOURNALING

WHO IS IT FOR?

TMS is available for use for Acquirers, Merchants and Arca Payments
